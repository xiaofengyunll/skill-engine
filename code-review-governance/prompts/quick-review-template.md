# Quick Review Template

用于普通 PR、小功能变更、缺陷修复和目标明确的局部审查。合并/发布结论统一按 `risk-assessment-template.md` 的 Merge Decision Rules 裁决。

## 结论

- 合并建议：`可以合并 / 暂缓合并 / 阻塞合并 / 可带债发布`
- 阻塞问题：`无 / 有`
- 最高风险等级：`致命 / 高风险 / 中风险 / 建议优化 / 无`
- 准入条件：`满足 / 部分满足 / 不满足 / 证据不足`
- 准出条件：`满足 / 部分满足 / 不满足`
